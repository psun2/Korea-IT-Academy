# DBeaver!

이클립스의 기본DatabaseDevelopment 환경의 빈약한 문제

- 최신 DBMS, 다양한 DBMS 지원 느림
- 쿼리 수행속도, 느림.
- 쿼리 Result 화면 빈약

DBeaver 로 극복 가능.

---

## SQL 파일 추가

> (ex) 오라클의 경우

CREAT - OTHER - SQL File
일단 생성하면 기본 이클립스 환경이 뜨는데, 닫고 다시 열어보자.

## 쿼리 실행

> (ex) 오라클의 경우

![이클립스 셋팅](./images/DBeaver이클립스셋팅1.png)

실행할 쿼리문 위에서
CTRL + ENTER (쿼리 한개 실행)

블럭선택후 해당 선택부분만
CTRL + ENTER 실행 가능
여러개 실행 할때는
여러 쿼리 선택후
ALT + X
